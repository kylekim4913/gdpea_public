Visual Studio Code (VS Code) 설치
다운로드: VS Code 공식 다운로드 페이지로 이동하여 "Mac" 버튼을 클릭해 .zip 파일을 다운로드합니다.
1. 다운로드한 .zip 파일(일반적으로 다운로드 폴더에 있음)을 찾습니다.
2. .zip 파일을 더블 클릭하여 Visual Studio Code 애플리케이션을 압축 해제합니다.
3. Visual Studio Code 애플리케이션을 응용 프로그램 폴더로 드래그하여 옮깁니다.
실행: 응용 프로그램 폴더에서 Visual Studio Code를 찾아 실행합니다.
파이썬 확장 설치: VS Code가 열리면 왼쪽 사이드바에 있는 확장 아이콘 (네모 네 개 모양)을 클릭합니다. 검색창에 "Python" (Microsoft사 제공)을 검색한 후 설치 버튼을 클릭합니다.
Python 3.13 설치
PC에 Python 3.13이 설치되어 있어야 합니다.
설치 확인:
python --version
`Python 3.13.x`가 나오지 않으면 Python 공식 다운로드 페이지에서 설치합니다.
터미널 열기 및 프로젝트 폴더 이동
VS Code에서 파일 > **폴더 열기...**를 선택하여 main.py 스크립트가 들어 있는 프로젝트 폴더를 엽니다.
상단 메뉴에서 터미널을 클릭한 후 새 터미널을 클릭합니다.
VS Code 화면 하단에 터미널 창이 열리며, 자동으로 프로젝트 디렉토리(ai_code 폴더 등)로 위치가 설정됩니다.
가상환경 생성하기 (venv)
활성화
.venv\Scripts\activate
비활성화
deactivate
패키지 설치 (requirements.txt)
활성화
pip install --upgrade pip
pip install -r requirements.txt
설치된 패키지 확인:
pip.list
이미지 분석 실행 (main.py)
프로젝트 내의 이미지 분석을 실행하려면 아래 명령어를 입력합니다:
python main.py
이미지가 자동으로 분석되어 결과 파일이 생성됩니다.
통계 분석 실행 (statics.py)
이미지 분석 결과를 기반으로 통계 데이터를 생성하려면 아래 명령어를 실행합니다:
python statics.py
연도별·형태별 키워드 빈도 및 통계가 자동으로 계산됩니다.

Install Visual Studio Code (VS Code)
Download: Go to the official VS Code download page and click the "Mac" button to download the .zip file.
1. Locate the downloaded .zip file (usually in your Downloads folder).
2. Double-click the .zip file to extract the Visual Studio Code application.
3. Drag the Visual Studio Code application to your Applications folder.
Execution: Find and run Visual Studio Code from the Applications folder.
Install Python Extension: When VS Code opens, click the Extensions icon (the four squares shape) on the left sidebar. Search for "Python" (provided by Microsoft) and click the Install button.
Install Python 3.13
Python 3.13 must be installed on your PC.
Check Installation:
python --version
If `Python 3.13.x` is not displayed, install it from the Python download page.
Open Terminal and Navigate to Project Folder
In VS Code, select File > **Open Folder...** to open the project folder containing the main.py script.
Click Terminal from the top menu, then click New Terminal.
A terminal window will open at the bottom of the VS Code screen, automatically setting the location to the project directory (e.g., the ai_code folder).
Create Virtual Environment (venv)
Activation
.venv\Scripts\activate
Deactivation
deactivate
Install Packages (requirements.txt)
Activation
pip install --upgrade pip
pip install -r requirements.txt
Check installed packages:
pip list
Run Image Analysis (main.py)
To run the image analysis within the project, enter the following command:
python main.py
The images will be automatically analyzed, and result files will be generated.
Run Statistics Analysis (statics.py)
To generate statistical data based on the image analysis results, run the following command:
python statics.py
Keyword frequency and statistics by year and format will be automatically calculated.
